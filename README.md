# 就活辞典【増補改訂 第二版】

辞典をひくように企業をしらべ、数字とことばで見比べる、就活生（26〜28卒想定）向けの企業辞典アプリ。
単体HTML＋ローカル画像だけで動きます（サーバー不要）。

## 開き方

- `index.html` をダブルクリック（Chrome / Edge / Safari 対応）
- スマホ幅・タブレット幅・PC幅すべてレスポンシブ対応

## フォルダ構成

```
就活辞典_最強版_v1\
├── index.html               … アプリ本体（見た目・動き。通常は触らない）
├── data.js                  … ★掲載データ（企業21社・お知らせ等。編集はここだけ）
├── SCHEMA.md                … データ項目の定義書（型・書式・能力値の付け方）
├── _データ検証.html          … データの自動チェック（編集後にダブルクリック）
├── CHANGELOG.md             … 変更履歴（編集したら1行追記）
├── pipeline\                … 企業追加の仕組み
│   ├── ADD_COMPANY.md       … 1社追加する手順書（AIリサーチプロンプト付き）
│   ├── fetch_logo.ps1       … ロゴ自動取得スクリプト（英語版Wikipedia）
│   └── REVIEW_CHECKLIST.md  … 監修チェックリスト（作った人と別の人が確認）
├── index_旧v1バックアップ.html … 決定版アップグレード前のバックアップ
├── index_旧v2バックアップ.html … データ分離（2026-07-05）前のバックアップ
├── manifest.webmanifest      … PWAマニフェスト（ホーム画面追加用・要ホスティング）
├── README.md                 … このファイル
└── assets\
    ├── logos\                … 掲載企業ロゴ（01_toyota.png 〜 21_hitachi.png）
    ├── icons\app_icon.png    … アプリアイコン
    └── hero_dictionary.png   … トップのヒーローイラスト
```

## 共同開発の進め方（二人で編集するときのルール）

1. **データは data.js だけを編集**する。見た目・機能の変更は index.html（相談してから）
2. 企業の追加は `pipeline/ADD_COMPANY.md` の手順で（リサーチ→追記→ロゴ→検証→監修）
3. 編集したら `_データ検証.html` を開いて**エラー0**を確認
4. **作った人と別の人**が `pipeline/REVIEW_CHECKLIST.md` で監修する
5. 変更のたび `CHANGELOG.md` に1行追記（日付・内容・担当）
6. 共有はフォルダごと（zip / Google Drive 等）。中は相対パスだけなので、どのPCでもそのまま動く

## 主な機能

- **さがす**: 検索（PCは Ctrl/⌘+K）・業界/職種/勤務地/初任給/福利厚生/エントリー状況/こだわり条件の絞り込み・五十音索引・並べ替え・グリッド⇄リスト切替・ページネーション
- **企業詳細**: 5タブ（会社概要/事業内容/働く環境/選考情報/社員の声）・六角レーダー・男女比・初任給・選考フロー・公式サイトリンク（`web`項目・青文字）
- **くらべる**: 2〜3社の能力値重ね図＋勝ち値ハイライト比較表
- **栞・あとで読む**: 気になる企業の保存（この端末のlocalStorageに保存）
- **エントリー記録**: 企業詳細で「インターン応募済／本選考応募済」の印をつけられ、カードに表示・絞り込みの「エントリー状況」で見返せる
- **やることリスト**: ES締切・面接日時などを企業と結びつけて登録（期限が近いと色つき表示・完了チェック・企業詳細の「＋やること」からも追加可）
- **付録**: 6つのものさしで並べ替える人気企業ランキング
- **PC右パネル**: きょうの注目3社（日替わり）／最近ひらいた項目／読了のしるべ
- **企業公認マーク**: 企業の担当者に内容を直接確認できた企業には `official:true` を付けると社名横に✓が出る

## データについて

- 掲載の数値は公開情報をもとにした**参考値**です。最新・正確な情報は各社の公式サイト（採用情報・IR）を必ず確認してください
- 能力値（6項目）・星評価は各種公表データをもとに編集部が指標化したものです
- 項目の意味・書き方は `SCHEMA.md` を参照

## 企業ロゴの取り扱い（重要）

`assets\logos\` の企業ロゴは各社の商標です（Wikipedia収載の公式ロゴより取得）。
**社内検討・提案用プロトタイプの用途に限定**してください。
サービスとして一般公開する場合は、各社の商標・ロゴ使用規定の確認（または掲載許諾）が必要です。
