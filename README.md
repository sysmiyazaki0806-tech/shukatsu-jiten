# 就活図鑑【増補改訂 第二版】

図鑑をひらくように企業をしらべ、数字とことばで見比べる、就活生（26〜28卒想定）向けの企業図鑑アプリ。

## 📱 アプリを使う（公開URL）

**https://sysmiyazaki0806-tech.github.io/shukatsu-jiten/**

スマホ・PCどちらもこのURLを開くだけ。スマホでは**アプリとしてインストール**できます：

- **iPhone**：Safariで開く → 下の共有ボタン（□↑）→「**ホーム画面に追加**」
- **Android**：Chromeで開く → 右上メニュー（⋮）→「**アプリをインストール**」（または「ホーム画面に追加」）

ホーム画面のアイコンから起動すると全画面のアプリとして動き、一度ひらいた内容は**電波が無い場所でも見られます**（オフライン対応）。

## ほかの開き方

- ローカル開発用：`index.html` をダブルクリックでもそのまま動く（Chrome / Edge / Safari 対応）
- スマホ幅・タブレット幅・PC幅すべてレスポンシブ対応

## 共同開発の進め方（二人で編集するときのルール）

このリポジトリ（GitHub）が**唯一の本番置き場**です。ここを更新すると1〜2分で全員のアプリに反映されます。

### はじめる前に（1回だけ）

編集する人はGitHubアカウント（無料）を作り、リポジトリのオーナーから**招待**を受けてください。
（オーナー側の操作：リポジトリの Settings → Collaborators → Add people で相手のGitHubユーザー名を入力）

### かんたん編集（ブラウザだけで完結・インストール不要）

1. GitHubで [`data.js`](https://github.com/sysmiyazaki0806-tech/shukatsu-jiten/blob/main/data.js) を開く
2. 右上の**鉛筆マーク（Edit）**を押して編集（企業データの追加・修正はこのファイルだけ）
3. 書き方に迷ったら `SCHEMA.md`（項目の定義）と `pipeline/ADD_COMPANY.md`（1社追加の手順）を見る
4. 編集を終えたら右上の緑ボタン **Commit changes** → 変更内容を1行書いて確定
5. 1〜2分待ってアプリのURLを再読み込み → 反映を確認
   （検証ページもURLで開けます：[`_データ検証.html`](https://sysmiyazaki0806-tech.github.io/shukatsu-jiten/_データ検証.html) で**エラー0**を確認）

### 品質を守るルール

1. **データは data.js だけを編集**する。見た目・機能の変更は index.html（相談してから）
2. 企業の追加は `pipeline/ADD_COMPANY.md` の手順で（リサーチ→追記→ロゴ→検証→監修）
3. 編集したら `_データ検証.html` で**エラー0**を確認
4. **作った人と別の人**が `pipeline/REVIEW_CHECKLIST.md` で監修する
5. 変更のたび `CHANGELOG.md` に1行追記（日付・内容・担当）
6. 間違えても大丈夫：GitHubに全履歴が残るので、いつでも前の状態に戻せます

## 主な機能

- **さがす**: 検索（PCは Ctrl/⌘+K）・業界/職種/勤務地/初任給/福利厚生/エントリー状況/こだわり条件の絞り込み・五十音索引・並べ替え（相性順あり）・グリッド⇄リスト切替・ページネーション
- **相性マッチ度**: プロフィールで「重視すること」（給与・安定など3つまで）を選ぶと、全企業にあなたとの相性%が表示される。カード・企業詳細・並べ替え・おすすめ欄に反映（能力値=編集部指標の平均＋志望業界一致の加点。参考値）
- **企業詳細**: 5タブ（会社概要/事業内容/働く環境/選考情報/社員の声）・六角レーダー・男女比・初任給・選考フロー・公式サイトリンク（`web`項目・青文字）
- **くらべる**: 2〜3社の能力値重ね図＋勝ち値ハイライト比較表
- **栞・あとで読む**: 気になる企業の保存（この端末のlocalStorageに保存）
- **エントリー記録**: 企業詳細で「インターン応募済／本選考応募済」の印をつけられ、カードに表示・絞り込みの「エントリー状況」で見返せる
- **わたしのメモ（選考メモ帳）**: 企業詳細に志望動機の下書き・面接で聞かれたこと・OB訪問メモなどを自由記入（自動保存・この端末のみ）。メモがある企業はカードに金の「メモ」印
- **閲覧・検索の履歴**: 自分が調べた企業・検索した言葉・見た業界を時刻つきで見返せる。タップで再検索／再表示、個別削除・全消去も可。**履歴は端末内（localStorage）のみに保存され、外部には一切送信されない**
- **共有カード**: 企業詳細の共有ボタンから、その企業の要点（社名・キャッチ・主要4データ・能力値レーダー・相性%）をまとめた画像カードを生成。スマホは共有シートでLINE等へ、PCは画像保存。**カードに企業ロゴ・個人メモは含まれない**（商標・プライバシー配慮）
- **系列（ブランド・グループ会社）**: 事業タブに主なブランド・グループ会社を表示。**ブランド名でも検索できる**（「ユニクロ」→ファーストリテイリング、「ドコモ」→NTT）
- **コンシェルジュ**: 「年収800万くらいで、東京で働けて、社員の仲がいい会社」のように希望をことばで書くと、条件を読み取っておすすめ3社＋理由を返す相談窓口。**端末内で完結（外部送信なし）**
- **ログインボーナス**: 毎日ひらくと連続日数を記録。節目で称号（三日の読者〜百日の図鑑マスター）。連続日数は「今日の就活占い」に表示
- **V-careerイベント連携**: トップにイベント告知バナー＋「イベント・説明会」ページに告知カード。出展企業には「V-career出展」の印と詳細内の申込導線。**イベント情報は `data.js` の `VC_EVENTS` を編集**（開催日を過ぎると自動非表示・申込URLは後から追記可）
- **選考ダッシュボード**: 応募企業の選考状況を「検討中→エントリー→ES提出→適性検査→面接→最終面接→内定／見送り」で管理。企業詳細の「選考ステータス」で段階をえらぶと、ステージ別カンバン＋ファネル集計に集まる。各カードに次のやること（期日つき）と「進める」ボタン。**この端末のみ保存・外部送信なし**
- **やることリスト**: ES締切・面接日時などを企業と結びつけて登録（期限が近いと色つき表示・完了チェック・企業詳細の「＋やること」からも追加可）
- **選考カレンダー**: やることリストを「リスト⇄カレンダー」で切替。月間カレンダーに種類別の色（ES締切=赤・面接=緑・説明会=青など）で表示され、日をタップするとその日の予定一覧＋「＋この日に追加」
- **業界図鑑（付録）**: 7つの業界を図鑑の見開き風に解説（ひとことで言うと・どんな業界か・主な職種・向いている人・よくある誤解・この図鑑に載っている企業一覧・関連キーワード）。企業詳細の業界タグからも飛べる。業界の解説は `data.js` の `INDUSTRIES` を編集（掲載企業は自動連動）
- **就活用語集（付録）**: ES・ガクチカ・固定残業代・オワハラなど、就活のことば52語を「ひとことで→くわしく→MEMO→関連語」の順で解説。カテゴリ絞り込み（書類・応募／選考・面接／イベント／内定・入社／働き方・待遇／基礎知識）＋用語内検索＋五十音の行見出し、関連語タップでジャンプ。用語は `data.js` の `GLOSSARY` を編集（書式は `SCHEMA.md`）
- **付録**: 6つのものさしで並べ替える人気企業ランキング
- **今日の就活占い**: 日付×おなまえで毎日変わる就活運（開運アクション・ラッキー企業つき）。文言は `data.js` の `FORTUNE` で編集可
- **就活タイプ診断**: 12問で16タイプ判定。結果から「重視すること」をワンタップ設定でき、相性%につながる。質問・タイプ文言は `data.js` の `SHINDAN` で編集可
- **PC右パネル**: きょうの注目3社（日替わり）／最近ひらいた項目／読了のしるべ
- **企業公認マーク**: 企業の担当者に内容を直接確認できた企業には `official:true` を付けると社名横に✓が出る

## データについて

- 掲載の数値は公開情報をもとにした**参考値**です。最新・正確な情報は各社の公式サイト（採用情報・IR）を必ず確認してください
- 能力値（6項目）・星評価は各種公表データをもとに編集部が指標化したものです
- 項目の意味・書き方は `SCHEMA.md` を参照

## 企業ロゴの取り扱い（重要）

`assets\logos\` の企業ロゴは各社の商標です（Wikipedia収載の公式ロゴより取得）。
**社内検討・提案用プロトタイプの用途に限定**してください。
サービスとして一般公開する場合は、各社の商標・ロゴ使用規定の確認（または掲載許諾）が必要です。
※このリポジトリ・URLは技術上は公開状態ですが、上記の理由から**URLの周知は社内メンバーに限定**する運用とします。
